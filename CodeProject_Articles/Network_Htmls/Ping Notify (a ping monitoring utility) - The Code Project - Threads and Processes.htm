Ping Notify (a ping monitoring utility)
By R%S.

The Ping Notify utility allows the user to define an IP address or URL set the time and start monitoring. The user may choose between beep on ping failing or on success.

Introduction

Simple and easy to use ping utility, was created in a short period and is using a wonderful neat gauge utility made by A.J.Bauer thanks A.J.

Background

I needed this tool because I wanted to know one morning when does a colleague of mine is coming to work (he docks his laptop when he arrives) so I created this tool to get a beep when he arrives. Since then I used it for some other tasks.

Using the code

To use this tool, compile using Visual Studio 2005, set the address, set the timeout, choose whether you would like beep sound on success or fail and click the Start button.

Points of Interest

It took me a long time to figure out how to use the AGauge in a WinForm. It is relatively very simple:

  • Display your form
  • Open the Tollbox
  • Click on Choose Items
  • Browse and select the AGauge.dll
  • Now drag a gauge into your form
