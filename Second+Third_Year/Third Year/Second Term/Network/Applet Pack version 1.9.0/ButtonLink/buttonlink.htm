ButtonLink

 

The ButtonLink applet provides a buttonlink. The page linked can be open in the same window or in a new one.  The size of the button is set to the size of the applet.                                  

 

             new page in a new window         

 

          new page in the same window

You can parameter :  the text of the button, the link, the location of the linked page (new or same window), and the size by setting the applet size.
