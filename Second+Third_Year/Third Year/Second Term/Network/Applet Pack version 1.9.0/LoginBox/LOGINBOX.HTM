LoginBox

 

This applet provides a small login window which purpose is to restrict access of a part of your web site. Allowed users need a login and a password, you can set one URL per login/password. The applet size is 2x2 or something like that because the grey applet zone is no use for this applet.
All the logins, passwords and URLs are in a hidden text file.

To try it: enter 'spoon' as login and 'cuillere' as password, you will access to the page 'secret_page.htm'.
              enter 'manga' and 'manga' and you will access yahoo.com (of course yahoo is not a secret web site ! ;))
