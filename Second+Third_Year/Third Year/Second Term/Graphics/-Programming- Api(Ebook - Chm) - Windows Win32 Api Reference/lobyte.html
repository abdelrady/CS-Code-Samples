LOBYTE Macro

Public Function LOBYTE(ByVal wValue As Integer) As Byte
	LOBYTE = Val("&H" & Right("00" & Hex(wValue), 2))
End Function

Description & Usage

LOBYTE extracts the low-order byte of a 16-bit integer (a word).

Return Value

The macro returns the low-order byte of the value passed to it.

Visual Basic-Specific Issues

Although it may seem unusual to use string operations for what would otherwise be a mathematical computation, it is necessary to implement LOBYTE this way in VB. Other methods to perform this task can fail for large inputs because VB interprets them as negative values, which causes an equivalent mathematical computation to return an unwanted value.

Parameters

wValue
The 16-bit integer to extract the low-order byte of.
