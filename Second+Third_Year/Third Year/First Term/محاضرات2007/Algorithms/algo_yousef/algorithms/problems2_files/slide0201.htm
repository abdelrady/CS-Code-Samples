wGiven n integers from 1 to k show how to count the number of elements from a to b in O(1) time with O(n + k) pre-processing time. As shown in COUNTING-SORT we can produce an array C such that C[i] contains the number of elements less than or equal to i. Clearly, C[b] - C[a] gives the desired answer.
