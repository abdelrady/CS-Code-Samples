Title: Snake World- Guarenteed Best Snake Game Ever
Description: The most entertaining, challenging and playable snake game ever. The game supports upto 4 players. There are 4 different game types in the snake world that are level based game, time attack, survival and king of the ring. U can fully customize the look of the game and the controls of each player in the most interactive manner. The game also introduces the concept of manual speed up/down in a snake game for the first time. The game introduces a lot of new and entertaining ideas, that are never introduced before in any snake game. The game offers 10 levels, each one of which offers the increased difficulty and fun. For more details, read the "Help Topics" portion under the Help Menu in the game.
