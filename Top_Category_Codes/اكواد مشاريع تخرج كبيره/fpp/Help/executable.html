Making Executable


You can even make an executable from you algorithm. Select "Make Executable" from "File" menu. Enter full path name for your algoritm.

The system does not generates a real executable, it just copies the program file and creates a special ".INI" file that is used to load and run the program without showing the flow. All functions that are used in your program should be copied to the same folder when the executable file is.

Usually, executable file is created for programs that have some other use except just showing the algorithms, for example "d.fpp" in examples - it is an image mapper for the web pages. It requires these files to be copied to the same folder where the executable (.exe file) is:
D.fpp - main algorithm with window.
D0.fpp ... D6.fpp - seven functions that are called as events by the main window (D.fpp).

Happy coding!
