Instructions on Password Recovery (takes about 10 minutes)

Perform the following on any Windows computer with internet access:

  1. Download this file. (0.2 MB). When prompted click Open or Run from current..., pressing Yes to any security warnings. (If you are unsure of its integrity, virus scan the disk first)
  2. When the download completes a window will appear. Click OK.
  3. Insert a blank floppy disk and click OK.
  4. Remove the disk when it's done.

Perform the following on the computer you want to recover passwords from:
(Windows NT/2000/XP/2003/Longhorn):

  1. Insert the disk THEN turn the computer on.
  2. Text will flick across the computer screen then the computer should switch off by itself. (Takes about two minutes)
  3. Remove the disk. (The computer will not have been changed in any way and will work normally again when the disk is removed)

Perform the following on any Windows computer with internet access:

  1. Insert the Disk.
  2. Type A:\UPLOAD.TXT (type this exactly) into the box below.
  3. Provide your email address so you can be contacted when the passwords have been recovered.
  4. Press Submit.
  5. Within minutes you should receive an email notifying you that the passwords have been processed.
  6. Check progress of recovery anytime by selecting the results tab
  7. If using the free service you will have to wait 48 hours for your passwords to be displayed to you. You can opt to use the PRIORITY SERVICE where we provide you with a release code which will display your passwords in about 10 minutes.

Advanced Options

If you have your own password hashes in pwdump2 format then you can upload them by pressing browse and then following Step 3 Part III onwards. Alternatively you can paste pwdump2 format information in the CD box below.

You can get hashes in pwdump2 format by using many utilities. Some are SAMInside, PWDUMP2, PWDUMP3, and network sniffer tools.

Using a CD Image

If you don't have a floppy drive, a CD version is available. To use it download this zip file (around 0.2 MB) and extract the ISO file and burn the ISO image to a CD. (Check your CD burner software for how to burn an ISO image file). Put the CD into the computer you want to recover passwords from and switch it on. Text will flick across the screen and you will theb be prompted to press any key. Do as it says and copy down the text that appears on the screen into the box below EXACTLY. You may have to press another key to view another screen full. If windows is installed more than once on the same computer then you will get a password list for each install. You only need to type in pairs of lines for users that you wish to recover passwords for.

An example is provided:

Administrator:500:0,CC,5E,9A,CB,AD,1B,25,C9,AA,D3,B4,35,B5,14,04,EE:
_99,6E,67,60,CD,DD,88,15,A2,C2,4A,11,0C,F0,40,FB,087CA:::

There should be no spaces and there should be two lines (remove any spaces that may be present). The Username must be entered as well. Make sure you put all the colons in the right places.

When you've entered the text EXACTLY, enter your email address in the lower box and press Submit
