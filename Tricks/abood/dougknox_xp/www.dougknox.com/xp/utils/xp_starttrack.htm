Windows XP Startup Tracker v3.8 - Updated 10/31/2003
 Copyright 2003 - Doug Knox

Version 3.8 - Added Options for what Startup Vectors to scan (Licensed version).

Version 3.7 - Added support for enhanced licensing procedure.

Version 3.1 - Added support for Windows 2000 (Visual Basic 6 Runtimes required).

This small GUI (Graphical User Interface) utility will check the Start Menu and the System Registry for items that load at startup.  It will also check for Disabled Startup items and changes to the default "Shell" value.  Support for listing all running Processes and Services has been added.  To use, just extract StartupTracker3.exe and the README.TXT file from the ZIP file and double click the EXE file.  You can create a log file each time its run, or choose to create the log file automatically.  The contents of the window are automatically copied to the Windows Clipboard, and you can copy and paste them manually.

This utility has been tested with Windows 2000, Windows XP and Windows Server 2003.  Requires VB6 Runtime Library (installed on XP/Server 2003 by default). It does not work with Windows 95, Windows 98 or Windows Millenium.
