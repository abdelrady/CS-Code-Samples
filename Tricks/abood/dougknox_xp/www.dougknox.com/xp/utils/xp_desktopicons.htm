XP_DesktopIconRename.exe - Rename System (Desktop) Icons
© Doug Knox - 2003

This small VB 6 utility will allow you to rename the My Computer, My Documents, My Network Places, Recycle Bin and the default Internet Explorer on the Desktop.  This is a per user setting.

To use: Download the XP_DesktopIconRename.zip file and save it to your hard drive. Double-click the XP_DesktopIconRename.zip file and extract XP_DesktopIconRename.exe to your hard disk.  To run the EXE just double click it, there is no installer.  You will be shown the current names of the 5 desktop icons that can be renamed with this utility.

Operating Systems: Windows XP only.  Requires VB 6 Runtime Libraries, included in Windows® XP.
