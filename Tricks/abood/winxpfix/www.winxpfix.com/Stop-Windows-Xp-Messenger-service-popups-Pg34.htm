    How to - Stop Windows XP Messenger Service Popup's
    By Raymond Garcia, Window Fix Zone.com

    The new spamming technique Messenger Spam, Broadcast Spam which delivers small text ads messages via the Microsoft Windows Messenger service. This is service is unrelated to the Microsoft MSN Instant Messenger application (MSN Messenger) used for sending instant messages to your friends.

    The Microsoft Windows Messenger service is meant to provide a means for the system administrators to broadcast simple text messages to one or more users on a local area network (LAN).

    Example, "The server will be shutdown in 15 minutes, please save all your open files and logout ASAP, Thank you.". Although the Messenger service way intended to be used only on your local LAN But, if you connect your system to the Internet, without using firewall protection, anyone on the Internet can transmit a Messenger popup (Broadcast Spam) to your LAN system, and on to your PC.     • Disable Messenger Service:
    By Raymond Garcia, Window Fix Zone.com

    You don't need to disable Messenger Service if you have installed a firewall that will stop remote users from sending you Messenger popup ads. But, if you don't need it, then disable it.

    If don't have a firewall installed or you plan not to install a firewall you can disable the Microsoft Messenger Service, But, this may leave you exposed to other security issues, so disabling Messenger is not recommended.

    Some software applications anti-virus, UPS system, SQL job Scheduler, and others use Messenger alerts to notify you of a critical system events. Use a firewall/router to block the Messenger Pop-ups and leave Messenger service enabled if you are using such products. WinXpFix.com

    Here are steps for disabling the Messenger service on Windows 2000 and XP systems:

    Windows 2000:
    * Click: Start/Settings/Control Panel
    * Open: Administrative Tools
    * Open: Services
    * Open: 'Messenger' Service
    * Click: Stop button
    * Change 'Startup Type' to DISABLE
    * Click OK to close everything
    * WinXpFix.com

    Windows XP:
    * Right-click: 'My Computer' icon and select 'Manager' Or Start> Settings> Control Panel> Administrative Tools> Computer Management.
    * Open: Services and Applications
    * Open: Services
    * Open: 'Messenger' Service
    * Click: Stop button
    * Change 'Startup Type' to DISABLE
    * Click OK to close everything.
    * WinXpFix.com
