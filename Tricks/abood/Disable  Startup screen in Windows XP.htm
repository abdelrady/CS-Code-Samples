To disable Windows XP startup screen

Disabling the Windows Startup Screen when your computer boots, will allow you to see what's really going behind it.


1) Click on Start and right click on My Computer ----> Select 'Properties' ----> Select the 'Advanced' tab

2) In the 'Startup and Recovery' section, click 'Settings'

 

3) In the 'System Startup' section, click on 'Edit' to open the boot.ini file in Notepad. You will see a line that looks similar to the one below:

multi(0)disk(0)rdisk(0)partition(1)\WINDOWS="Microsoft Windows XP Professional" /fastdetect

4) To disable the Startup Screen change the line to read as below:

multi(0)disk(0)rdisk(0)partition(1)\WINDOWS="Microsoft Windows XP Professional" /fastdetect /sos

5) Close Notepad, clicking ----> Yes when Notepad asks if you want to save your changes

6) Click ----> OK and exit System Properties. Reboot your computer and the Startup Screen will be gone
