How to: Determine Which Control Has the Focus (Visual C#)
See Also  Example






This example displays the name of the currently selected Windows Forms control in a Label control.

Example

 Copy Code
private void DisplayNameOfActiveControl()
{
    label1.Text = this.ActiveControl.Name;
}

Compiling the Code

This example requires:

  • A form with a Label control named label1. Add a control, such as a Button or a TextBox, to the form and place a call to DisplayNameOfActiveControl in the label1_Click event. When you click the Label, it displays the name of the control that has the focus.
