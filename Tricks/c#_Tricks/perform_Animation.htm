How to: Buffer the Display to Reduce Flicker (Visual C#)
See Also  Example






This example demonstrates how to reduce flicker by performing drawing operations into a buffer, before copying the buffer to the display. This technique is particularly useful when performing animation.

Example

 Copy Code
// Define a pen
System.Drawing.Pen myPen;

// Obtain the form's Graphics object.
System.Drawing.Graphics formGraphics = this.CreateGraphics();

// Define the bitmap for the buffer.
Bitmap bufferBitmap = new Bitmap(this.Width, this.Height);
System.Drawing.Graphics bufferGraphics = Graphics.FromImage(bufferBitmap);

// Draw into the buffer.
for (int x=0; x<300; x+=2)
    bufferGraphics.DrawLine(myPen, 0, 0, x, 300);

// Copy the buffer to the form.
formGraphics.DrawImage(bufferBitmap, 0, 0);

// Clean up resources.
formGraphics.Dispose();
bufferGraphics.Dispose();

myPen.Dispose();

Compiling the Code

This example requires a Windows Application project.

The code needs to be within the scope of the form class.

The instance of the form is represented by this.

Robust Programming

You should always call Dispose on any objects that consume system resources, such as Pen and Graphics objects.
