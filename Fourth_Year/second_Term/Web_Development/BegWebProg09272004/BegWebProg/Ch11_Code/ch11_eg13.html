You can do new things with CSS positioning that designers are only just learning.CSS approaches positioning of elements on a page in a completely different manner than tables. Each element is treated as a box that is positioned in terms of its containing element. These boxes can then be placed on the canvas relative to their containing element, rather than just in grids, so you can create overlapping effects.
