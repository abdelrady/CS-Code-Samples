What is Smart Door Access System?

- System centralized on solving the problem of entrance control to facilities, banks or any other secure corporation. Such system manages card readers, attached to each door to the lab, and opens the doors for registered students.

- There is a server computer that holds the lists of doors and students; it has a graphical interface that can be accessed over the network. The system supports multiple doors, multiple students.
