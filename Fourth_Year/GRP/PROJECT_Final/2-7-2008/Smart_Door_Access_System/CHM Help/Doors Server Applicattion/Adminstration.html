Tools Menu Item: Adminstration

This menu item allows server administration employees to change their passwords.

This form has element that allow user to change user name and password.but to do this he should has the current user name/password before change ocurred as shown in fig (23b )

Figure (23b) show  form that allow user to change user name and password


Explain How to use Adminstration-User Profile

We will explain the content of figure(23b).

1- Current Data: have two content:

User Name

Enter your current user name.

Passward

Enter your current passward.


2- Change Profile Data : have Tree content:

New User Name

Enter your new user name you want to type.

New Passward

Enter your new passward you want to type.

Retype Passward

Re-enter your new passward you write in privous text field.


3-We have Three Buttons in the bottom of fig(23b)
** if you want to Save your change press on
"Apply" Button

which save the change only.
"OK" Button

which save the change and exit the window.
** if you want to Cancel your operation press on
"Cancel" Button

which don't save your change.
