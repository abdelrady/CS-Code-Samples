Application Menu

The Application Menu divide into three menu as we show in the previous figure figure (21)

  • Server menu
  • Tools menu
  • Help menu

Server Menu item


It has 4 menu items which include :

  • Disconnect server:which disconnect server socket so drop all clients found in doors list tab.
  • Close database Connection: this menu item allows you to close local database connection.
  • Restart server:this button allow you to restart server application.
  • Exit: allow you to close server application.

Figure (22) show server menu that contain mandatory items

Tools Menu Item:

It has 2 menu items which include :(* you must press in that two links to know there details)

  • Server setting
  • Administration

Figure (23) The tools menu which contain program and user configuration

Help Menu item:

it has 2 items which include:

  • Help Topics:this menu item opens application help file (help.chm) that contains help topics of our application.
  • About us:this menu item show form contains Application development team information.
