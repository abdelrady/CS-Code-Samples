Clients action menu

This menu (open door,monitor door,restart remote application and more...) is used to communicate with online clients triggering them to execute specific actions such as send its information to server,send desktop images to server so server will support remote connection and control. As shown in fig (27)

Figure (27) show client action menu so we can monitor door or open door remotely.

Explain How to use Client Action Menu

We have five content in this menu.

Open Door

Open the door you will press in it .

Monitor Door

Will explian later ( Desktop Monitor ).

Restart Remote Application

Will colse and open your remote connection.

Disconnect Door Socket

Will make this door diconnect from connection .

Properties

Give to you details of that door .
