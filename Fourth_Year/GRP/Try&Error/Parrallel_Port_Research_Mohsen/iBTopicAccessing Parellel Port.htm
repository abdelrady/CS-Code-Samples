Hmmm...
The config seems quite okay.

In your config, you have these two lines:
CODE
serial0.hardwareFlowControl = "TRUE"
parallel0.bidirectional = "TRUE"

Try changing them to this:
CODE
serial0.hardwareFlowControl = "FALSE"
parallel0.bidirectional = "FALSE"

Also, if you are already using your parallel and serial port for something else in Windows, it may be that the ports are busy.

Finally, if you are not using these ports in your virtual machine (you just get some annoying error messages), you can disable them altogether:
CODE
serial0.present = "FALSE"
parallel0.present = "FALSE"

If this doesn't work at all, you could try reinstalling VMware Player.

QUOTE
I've installed vmware on my pc... windows xp..

Ah, you're using SuSE Linux...
Your first post made me believe that you were running Windows XP.
Are you absolutely sure that your serial and parallel ports are working in Linux?

If they are not working in Linux, VMware will not be able to use them either.
VMware can only (with the exception of some USB devices) use hardware that is correctly configured in the host operating system.

If you really need access to serial and parallel ports from your virtual machine, make sure they are correctly configured and working in Linux.

If you don't need these ports, just disable them in the config. Just as I told you in my earlier post.
