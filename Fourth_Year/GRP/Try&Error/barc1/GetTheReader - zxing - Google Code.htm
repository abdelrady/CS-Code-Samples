How to get a compiled reader for your phone

Get The Reader

Before we show you how to get it, note that only certain phones can currently run this software. This is also a very early build, and is suitable for developers and the adventurous, who are willing to put up with some glitches. Don't be too disappointed if you try this and it gives an error (but please report it to us). This probably not baked enough to send off to grandma to download onto her phone yet. That said...

Requirements

  • A camera (obviously)
  • Auto-focus capability, and preferably macro mode or optical zoom
  • Support for Java applications (developers: MIDP 2.0 and CLDC 1.1)
  • (Optional but pretty important) Support for advanced multimedia functionality (developers: MMAPI and AMMS / JSR-234)
  • (Optional, but needed to resolve URLs) Internet access from the phone

Supported Phones

This list is still under construction, as we have not had an opportunity to test on many devices. We believe it will work on devices like these below (and those in the same "family"):

  • Sony Ericsson K550, K610, K630 K790, K800, K810, K850, T650i, V640, W580, W610, W660, W710, W830, W850, W880, W910, Z610, Z750
  • Nokia 6500, 7500, 7900, 6260, E65
  • Motorola RAZR V3x (?)

Please let us know what your experience is.

Partially Supported Phones

Here are a few that appear to be confirmed to not support this application, but please see below for an alternate version that may run and provide limited functionality:

  • Sony Ericsson K510, K750, W810
  • Nokia N70, N73, N80, N91, 6300
  • N95 (installs, but no focus control?)

Unsupported Phones

  • any Windows Mobile device
  • anything without a camera or Java, of course
  • Devices on Sprint network? seems these need signed applications, but need more testing
  • Sony Ericsson M600i
  • Nokia 6620

Where do I get it?

Navigate to this address in your phone's browser, if you're sure you want to try it:

http://zxing.org/w/

(The application URL, directly, is: http://zxing.org/w/ZXingReader.jad)
