Label Definitions

A Label Definition consists of a line code, an x position, a y position, and a string information, each separated by a pipe ('|') character. A Label Definition can also contain setup information, detailing the label orientation for printing, the label size, and the font used on the label.

Format

{line code}|{x}|{y}|{information}

Line Codes

s - Indiates settings for the label. Setting statements contain special information, as detailed below.

f - Draw a frame around the outside of the label. No other information is required when using this line code.

b - Draw a barcode, with the top left corner at the given coordinates. Also, print the text equivalent immediately below the barcode.

t - Print the given text, with the top left corner at the given coordinates.

x And y Positions

The coordinates are in number of pixels from the left for the x position, or from the top for the y position, respectively.

The x position can be from -[width] to [width], where [width] is the actual width of the useable space on the label. This value can also be the character "c", which indicates that the {information} will be centered across the label. A negative value indicates that the {information} is indented from the right side of the label, i.e. right justified. Setting this value to either extreme means that the {information} may not be viewable. Note: For drawing a barcode, use only values from 0 to [width], as the barcode cannot be right justified.

The y position can be from 0 to the actual height of the useable space on the label. However, setting this value to the height of the label means that the {information} may not be viewable.

Information

The {information} can contain text and/or variables. Variables are indicated by a leading and trailing tilde ("~"), and will be replaced by data when previewing or printing the label. For example, "~date~" is a valid data variable.

Settings

Setting statements should always be at the top of the label definition. There are four (4) types of setting statements: barcode format, orientation, size, and font.

Barcode Format

The barcode format setting statement has one possible entry. By default, Code 39 is assumed, but Interleaved 2 of 5 can be used by inserting the following statement into the definition file:
  s|c|I25

Note: Interleaved 2 of 5 barcodes only support numeric values, where Code 39 supports nearly all characters.

Orientation

The orientation setting statement has one (1) item, which indicates if the label should be printed in portrait or landscape mode. The two (2) possibilities for an orientation setting statement are as follows:
Portrait= s|o|p
Landscape= s|o|l

Default: If this statement is not included, the label will be printed in portrait mode.

Size

The size setting statement must be in the following format:
  s|s|{scale}|{scale height}|{scale width}

The {scale} value must be one (1) of the following: "in" (inches), "pix" (pixels), or "cm" (centimeters).

The {scale height} value must be numeric, and indicates the actual useable height of the label, in the given scale.

The {scale width} value must be numeric, and indicates the actual useable width of the label, in the given scale.

Default: If this statement is not included, the default values will be used. The default scale is inches, with scale height set to 1 inch and scale width set to 3 inches.

Font

The font setting statement must be in the following format:
  s|f|{font name}|{font size}|{bold}|{italic}

The font setting applies to all of the text on the label, but does not affect any generated barcodes.

The {font name} must be the name of a valid font. If the font is invalid, or the name is spelled incorrectly, the default {font name} will be used.

The {bold} value must be "true" or "false", where "true" indicates that all text will be bold.

The {italic} value must be "true" or "false", where "true" indicates that all text will be in italics.

Default: If this statement is not included, the default font setting will be used, which is Arial, size 8, Bold.

Sample

s|o|l
s|s|pix|85|285
s|f|Arial|8|true|false
f
b|c|10|Label Test
t|10|65|James Pollard
t|-10|65|~date~
